# 🖥️ Simple Computer Emulator

Программная эмуляция 15-разрядной архитектуры ЭВМ (фон-неймановская модель), включающая полнофункциональный процессор, подсистему памяти, консольный интерфейс на базе ANSI escape последовательностей и собственный транслятор Ассемблера.

## 🚀 Ключевые особенности

* **Core Engine:** Полная программная модель CPU (ALU + CU), 100 ячеек 15-битной оперативной памяти и регистровая модель.
* **Low-Level I/O:** Собственная реализация работы с терминалом через управляющие последовательности ANSI (без использования сторонних библиотек вроде ncurses).
* **Custom Assembler:** Транслятор из мнемоник simple assembler (.sa) в бинарный машинный код эмулятора.
* **Modular Architecture:** Проект разбит на независимые статические библиотеки, что обеспечивает высокую связность кода и низкую зависимость модулей.

## 🏗 Архитектура системы

- **Разрядность:** 15 бит (1 бит знака + 14 бит команды/данных).
- **Память:** 100 ячеек оперативной памяти (RAM).
- **Регистры:** - `Accumulator` (15 бит) — для арифметических операций.
    - `Instruction Counter` (7 бит) — адрес текущей команды.
    - `Flags` — регистр состояния (переполнение, деление на 0, выход за границы, игнорирование импульсов, неверная команда).

### Программные модули:
- `mySimpleComputer`: Управление RAM, регистрами и декодированием команд.
- `myTerm`: Управление буфером терминала (цвета, позиционирование курсора).
- `myBigChars`: Вывод псевдографических символов и шрифтов в консоль.
- `myReadKey`: Обработка пользовательского ввода в неканоническом режиме терминала.
- `simpleassembler`: Трансляция программы на языке simple assembler в машинные команды эмулятора

## 🛠 Технологический стек

* **Язык:** C.
* **Сборка:** GNU Make (многоуровневые Makefile).
* **CI/CD:** GitLab CI (автоматическая сборка и проверка clang-format).

## 📥 Запуск проекта

### Предварительные требования
Для работы требуется Linux-окружение (или WSL) и установленный компилятор `gcc`.

### Сборка и запуск

- Клонирование репозитория
```bash
git clone https://github.com/SperanskyArtyom/simple-computer.git
cd simple-computer
```

Полная сборка всех модулей и библиотек
```bash
make
```

Генерация шрифта bigchars
```bash
make font
```

Или
```bash
./console/font
```

Запуск основного приложения консоли
```bash
make run
```

Или
```bash
./console/console
```

Трансляция simple assembler программы (`program.sa`) в исполняемый файл simple-computer (`program.o`)
```bash
./simpleassembler/sat <program.sa> <program.o>
```
